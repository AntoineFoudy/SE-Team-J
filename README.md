To Set up and run this system you need to :
1. Have a IDE for Java, JDK 25
2. Have MySQL server and for it to be running
3. Change the password in BookStoreApp.java ( Line 46 ) and in DBClass.java ( Line 23 )

If these are in order the application should run fine

Please note for the amount of stock for each book that if you run the program multiple time, 
the stock will run out and since all the information is stored and updated in a database instead of memory the amount of stock will not reset everytime the application is started
